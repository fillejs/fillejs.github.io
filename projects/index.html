# 开源项目

  • gitmars (opens new window)：定制化的工作流操作工具
  • axios-ex (opens new window)：axios 扩展接口并发控制、请求取消、自动重试、请求数量限制
  • axios-q (opens new window)：axios 扩展工具解决接口并发问题，支持取消请求
  • egg-http-proxy-plus (opens new window)：基于最新的 http-proxy-middleware 开发的更加智能的的 egg 代理中间件，解决了代理文件上传接口的问题
  • egg-nuxt-tool (opens new window)：用于渲染 Nuxt 工程的 egg 中间件
  • rollup-plugin-replace-shebang (opens new window)：一个自动替换 shebang 的 rollup 插件
  • js-cool (opens new window)：采用 rollup + typescript 开发的 js 常用方法库
  • eslint-reporter (opens new window)：支持查看每个人名下 eslint 报错的工具，支持按姓名导出代码质量报告
  • eslint-config-sets (opens new window)：eslint 配置预设
  • clipboard-parser (opens new window)：剪贴板解析器，支持解析@RequestParam/@ApiModelProperty 接口定义代码、Word、Excel 以及其他表格类数据
  • postmessager (opens new window)：iframe postmessage 通信整体方案
  • nuxt3-element-plus-starter (opens new window)：nuxt3 element-plus starter
  • @saqqdy/prettier-config (opens new window)：个人自用的 prettier 配置
  • xterm-style (opens new window)：xterm 样式模板
  • marsgit (opens new window)：gitmars 的 deno 版本（开发中）
  • vuepress-doc (opens new window)：vuepress 文档模板
  • vue-lazy-compile-webpack-plugin (opens new window)：vue 懒构建插件
  • domtoimg (opens new window)：基于 dom-to-image 优化的版本，解决移动端转图分辨率过低的问题
  • ui-lib (opens new window)：vue 组件库完整方案
  • saqqdy-cli (opens new window)：saqqdy cli 工具
  • electron-express-starter (opens new window)：electron express starter
  • electron-vite-vue3-starter (opens new window)：electron vite vue3 starter
  • windicss-vite-vue3-starter (opens new window)：windicss vite vue3 starter
